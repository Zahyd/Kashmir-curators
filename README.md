# The Kashmir Curators

Professional travel management platform for Kashmir tour packages, hotels, and cabs.

## Project Overview

The Kashmir Curators is a comprehensive travel platform designed to provide curated travel experiences in the beautiful valleys of Kashmir. It features an administrative portal, sales portal, and customer-facing dashboard.

## Technologies Used

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, Prisma, PostgreSQL
- **Real-time**: Socket.io
- **State Management**: React Query

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd kashmir-connect
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Development

- Frontend runs on: `http://localhost:8080`
- Backend runs on: `http://localhost:5000`
